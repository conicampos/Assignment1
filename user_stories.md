# User Stories

## 1. Event Creation and Management
* **As an** Organizer, **I want to** create a new event with details such as title, description, date, time, location, and maximum capacity, **so that** users can see all the necessary information to attend.
* **As an** Organizer, **I want to** edit the details of my existing events, **so that** I can keep attendees informed if there are any changes in the schedule or venue.
* **As an** Organizer, **I want to** cancel an event, **so that** users are notified and no more registrations are accepted.

## 2. Event Discovery and Browsing
* **As a** User, **I want to** search and filter events by category, date, or location, **so that** I can easily find activities that match my interests and availability.
* **As a** User, **I want to** click on an event to view its full details, **so that** I can decide whether I want to participate or not.
* **As a** User, **I want to** see how many spots are left for an event, **so that** I know if I need to register quickly before it sells out.

## 3. Registration and Waiting List Behavior
* **As an** Attendee, **I want to** register for an upcoming event, **so that** I can secure my spot and receive confirmation.
* **As an** Attendee, **I want to** be automatically added to a waiting list if an event is already at full capacity, **so that** I can get a spot if someone else cancels their registration.
* **As an** Attendee, **I want to** cancel my event registration if I can no longer attend, **so that** my spot can be given to the next person on the waiting list.

## 4. Review and Rating of Past Events
* **As an** Attendee, **I want to** leave a star rating and a text review for an event I have participated in, **so that** I can share my experience and feedback.
* **As a** User, **I want to** read reviews and see the average rating of past events, **so that** I can assess the quality of events hosted by a specific organizer.

## 5. Administrative Actions (Moderation)
* **As an** Administrator, **I want to** remove events that violate the platform's guidelines, **so that** the application remains safe and appropriate for everyone.
* **As an** Administrator, **I want to** delete inappropriate or offensive reviews, **so that** the community environment remains respectful.
* **As an** Administrator, **I want to** suspend or ban users who repeatedly break the rules, **so that** the integrity of the platform is maintained.
